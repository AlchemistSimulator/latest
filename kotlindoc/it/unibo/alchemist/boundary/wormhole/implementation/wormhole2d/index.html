Wormhole2D

open class Wormhole2D<P : Position2D<P>?> : IWormhole2D<P>, Any

Partial implementation for the interface IWormhole2D.
I am considering the particular case of the view as an entity into the screen-space: the y-axis grows on the bottom side of the screen.
