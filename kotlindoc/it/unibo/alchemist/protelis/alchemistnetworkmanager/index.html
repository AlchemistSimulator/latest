AlchemistNetworkManager

class AlchemistNetworkManager: NetworkManager, Serializable

Emulates a NetworkManager. This particular network manager does not send messages instantly. Instead, it records the last message to send, and only when #simulateMessageArrival(double) is called the transfer is actually done.
