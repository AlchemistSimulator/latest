AlchemistExecutionOptions

data class AlchemistExecutionOptions: Any

Alchemist options provided by the command line.

Fields

Name Description
val configuration: String?

null if no simulation file is specified (default), file path otherwise
val headless: Boolean

true if forced, false otherwise (default)
val variables: List<String>

selected batch variables. Defaults to emptyList
val batch: Boolean

whether batch mode is selected.
val export: String?

the path to the file root of the data to export, or null if unspecified (export disabled).
val distributed: String?

the path to the file with the load distribution configuration, or null if the run is local
val graphics: String?

the path to the effects file, or null if unspecified
val help: Boolean

true if print help function is selected
val interval: Double

sampling time, defaults to defaultInterval
val server: String?

if launched as Alchemist grid node server, the path to the configuration file. Null otherwise.
val parallelism: Int

parallel threads used for running locally. Defaults to defaultParallelism
val endTime: Double

final simulation time. Defaults to defaultEndTime

AlchemistExecutionOptions

data class AlchemistExecutionOptions: Any

Alchemist options provided by the command line.

Fields

Name Description
const defaultInterval: Double

If no sampling interval is specified, this option value is used. Defaults to 1.0.
val defaultParallelism: Int

If no specific number of parallel threads to use is specified, this value is used. Defaults to the number of logical cores detected by the JVM.
val defaultEndTime: Double

Default final time to be used if no final time is specified. Defaults to Double.POSITIVE_INFINITY.
