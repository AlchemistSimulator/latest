RemoveJunctionInNeighbor

class RemoveJunctionInNeighbor: AbstractNeighborAction<Double?>

Represent the action of removing a junction between a neighbor and the current node.
This action only remove the junction reference inside the neighbor node, the current one totally ignore that a junction has been removed.
This is a part of the junction removal process.
See RemoveJunctionInCell for the other part of the process.
