BaseEuclideanNavigationAction

open class BaseEuclideanNavigationAction<T, N, E> : AbstractNavigationAction<T, Euclidean2DPosition, Euclidean2DTransformation, N, E, ConvexPolygon, Euclidean2DPassage>

An AbstractNavigationAction working with euclidean spaces. This action accepts an Euclidean2DEnvironmentWithGraph whose graph features ConvexPolygonal nodes and Euclidean2DPassages as edges.
