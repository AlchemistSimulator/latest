SAPEREGradient

class SAPEREGradient<P : Position<P>?> : AbstractReaction<MutableList<ILsaMolecule?>?>

This class provides a fast and stable gradient implementation, inspired on the NBR construct used in Proto.

Parameters

Name Description
env: Environment<MutableList<ILsaMolecule?>?, P>?

Environment<MutableList<ILsaMolecule!>!, P>!: the current environment
n: ILsaNode?

ILsaNode!: the node where this reaction is scheduled
sourceTemplate: ILsaMolecule?

ILsaMolecule!: a template ILsaMolecule representing the source
gradientTemplate: ILsaMolecule?

ILsaMolecule!: a template ILsaMolecule representing the gradient. ALL the variables MUST be the same of sourceTemplate: no un-instanced variables are admitted when inserting tuples into nodes
valuePosition: Int

Int: the point at which the computation of the new values should be inserted. All the data after this position will be considered "additional information" propagated by the source. All the values before this one, instead, will be used to distinct different gradients
expression: String?

String!: the expression to use in order to calculate the new gradient value. #T and #D are admitted, plus every variable present in the gradient before valuePosition, and every variable matched by the contextTemplate
contextTemplate: ILsaMolecule?

ILsaMolecule!: a template ILsaMolecule. It can be used to match some contents of the local node in order to have local information to use in the gradient value computation
gradThreshold: Double

Double: if the value of the gradient grows above this threshold, the gradient evaporates
td: TimeDistribution<MutableList<ILsaMolecule?>?>?

TimeDistribution<MutableList<ILsaMolecule!>!>!: Markovian Rate
