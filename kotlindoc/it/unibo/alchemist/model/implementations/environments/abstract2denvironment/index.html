protected open fun nodeAdded(node: Node<T>?, position: P, neighborhood: Neighborhood<T>?)

Subclasses may want to override this method to hook to the node addition event. Overriders should call the super implementation, as it ensures the environment bounds are updated considering the newly included object.

moveNodeToPosition

open fun moveNodeToPosition(node: Node<T>?, newpos: P)

Subclasses may override this method if they want to change the way a node moves towards some absolute position. Overriding this method will also influence #moveNode(Node, Position2D), as it calls this method in the current implementation

moveNode

open fun moveNode(node: Node<T>?, direction: P)

Subclasses may override this method if they want to change the way a node moves towards some direction. The current implementation internally calls #moveNodeToPosition(Node, Position2D), as such, overriding that method may suffice.

Parameters
