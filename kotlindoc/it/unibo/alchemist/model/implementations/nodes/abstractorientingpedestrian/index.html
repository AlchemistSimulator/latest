AbstractOrientingPedestrian

abstract class AbstractOrientingPedestrian<T, P, A : GeometricTransformation<P>, N : ConvexGeometricShape<P, A>, M : ConvexGeometricShape<P, A>, E, F : GeometricShapeFactory<P, A>> : OrientingPedestrian<T, P, A, N, DefaultEdge>, HomogeneousPedestrianImpl<T, P, A, F>

An abstract OrientingPedestrian. This class defines an algorithm capable of generating a pseudo-random cognitiveMap based on a NavigationGraph of the environment. The latter can be obtained from EnvironmentWithGraphs, which is the only type of environment this pedestrian can be placed into. The creation of landmarks is left to subclasses via factory method.

Fields

Name Description
open volatileMemory: MutableMap<ConvexGeometricShape<P, A>, Int>

The volatile memory of the agent: it models the ability to remember areas of the environment already visited by the agent since the start of the simulation. In particular, each area is paired with the number of visits. Areas are assumed to be represented as ConvexGeometricShapes, as in NavigationGraphs.
open cognitiveMap: NavigationGraph<P, A, N, DefaultEdge>

The cognitive map of the agent. It's composed of landmarks (elements of the environment easy to remember due to their uniqueness) and spatial relations between them. It is represented using a NavigationGraph.
val knowledgeDegree: Double

The knowledge degree of the agent concerning the environment, it's a Double value in 0,1 describing the percentage of the environment the agent is familiar with prior to the start of the simulation (thus it does not take into account the knowledge the pedestrian will gain during it, namely the volatileMemory).
protected randomGenerator: RandomGenerator

The random generator to use in order to preserve reproducibility.
