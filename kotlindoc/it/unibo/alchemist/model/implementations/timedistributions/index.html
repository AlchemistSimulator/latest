open class AnyRealDistribution<T : Any?> : AbstractDistribution<T>

This class is able to use any distribution provided by Apache Math 3 as a subclass of RealDistribution. Being generic, however, it does not allow for dynamic rate tuning (namely, it can't be used to generate events with varying frequency based on it.unibo.alchemist.model.interfaces.Condition#getPropensityContribution().
