open class AwtMutableConvexPolygon: AbstractConvexPolygon, MutableConvexPolygon

MutableConvexPolygon partly delegated to AwtEuclidean2DShape and java.awt.geom. Each modification operation on this object has a time complexity of O(n), where n is the number of vertices/edges. Collinear points are allowed.

fromShape

fun fromShape(shape: Shape): Optional<MutableConvexPolygon>

Creates a MutableConvexPolygon from a java.awt.Shape. If the Polygon could not be created (e.g. because of the non-convexity of the given shape), an empty optional is returned. Each curved segment of the shape will be considered as a straight line.
