Segment2D

data class Segment2D<P : Vector2D<P>> : Any

Defines a segment from first to second in an euclidean bidimensional space.

Fields

Name Description
val length: Double

The length of the segment.

Indicates if the segment is aligned to the x axis.

Indicates if the segment is aligned to the y axis.
val isAxisAligned: Boolean

Indicates if the segment is axis-aligned.
val slope: Double

The slope of the segment. If the two points coincide, this is Double.NaN.
val intercept: Double

The intercept of the line passing through first and second.

A segment is degenerate if its points coincide.

toRange

fun toRange(getXCoords: Boolean): ClosedRange<Double>

Maps the segment a ClosedRange, this is done by extracting either the X coordinates or the Y coordinates of the two endpoints of the segment. getXCoords indicates which pair of coordinates should be extracted. This can be useful e.g. to represent portions of axis-aligned segments without creating new ones.
