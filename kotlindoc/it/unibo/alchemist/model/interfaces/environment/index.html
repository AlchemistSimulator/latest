getNodesWithinRange

abstract fun getNodesWithinRange(center: Node<T>?, range: Double): ListSet?

Given a node (center) this method returns a list of all the surroundings nodes within the given range. Note that this method (depending on the implementation) might be not optimized and it's consequently much better to use Environment#getNeighborhood(Node) and filter the neighborhood if you are sure that all the nodes within the range are connected to the center.
