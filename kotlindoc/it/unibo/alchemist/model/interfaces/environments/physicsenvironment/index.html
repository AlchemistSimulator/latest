PhysicsEnvironment

interface PhysicsEnvironment<T, P, A : GeometricTransformation<P>, F : GeometricShapeFactory<P, A>> : EuclideanEnvironment<T, P>, Any

An environment supporting physics and nodes shapes. Note: due to the high number of parameters its's highly recommended to not use this interface directly, but to create an apposite interface extending this one instead.
