it.unibo.alchemist.loader.variables

package it.unibo.alchemist.loader.variables

Classes

Name Description
data class JSR223Variable<R> : DependentVariable<R>, Any

This variable loads any JSR-233 language available in the classpath.
class GeometricVariable: PrintableVariable<Double?>

A variable ranging geometrically (exponentially) in a range. Ideal for log-scale comparisons. e.g. a GeometricVariable with minimum = 1, maximum = 100 and samples = 5 will range over [1, ~3.16, 10, ~31.62 100]. Both min and max must be strictly bigger than 0.
class NumericConstant: DependentVariable<Number?>, Any

A numeric constant.
class ArbitraryVariable: PrintableVariable<Serializable?>

A variable spanning over an arbitrary set of values.
abstract class PrintableVariable<V : Serializable?> : Variable<V>, Any

A variable stub, with a default #toString() method.
class Flag: PrintableVariable<Boolean?>

This variable is a flag. Being booleans not a valid data type in charts, this variable just outputs 0 and 1. This is equivalent to a LinearVariable with two samples ranging from 0 to 1.
interface Variable<V : Serializable?> : Serializable, MutableIterable<V>, Any

A variable simulation value, that provides a range of values for batches, and a default value for single-shot runs.
interface DependentVariable<V : Any?> : Serializable, Any

A dependent variable, namely a variable whose value can be obtained given the values of other variables.
class LinearVariable: PrintableVariable<Double?>

This class represents a linear variable, namely a variable whose values span linearly between minimum and maximum.
