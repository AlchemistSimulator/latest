Classes

Name Description
abstract class AbstractCloseTo<T, P : Position<P>> : Displacement<P>, Any

A generic Displacement that displaces a certain nodeCount of nodes in the proximity of a number of sources. Higher variance implies higher dispersion. Subclasses must identify the sources
abstract class AbstractRandomDisplacement<P : Position<P>> : Displacement<P>, Any
class CloseToAlreadyDisplaced<T, P : Position<P>> : AbstractCloseTo<T, P>

This Displacement places new nodes in the proximity of those already included in the environment. Behaviour if there are no nodes already inserted is undefined.
open class Grid: Displacement<Position<*>>, Any

A (possibly randomized) grid of nodes.
class Point2D

Alias for using pairs as bidimensional points.
open class Polygon<P : Position2D<P>> : AbstractRandomDisplacement<P>

Creates a new Polygon with the given points.
class SpecificPositions: Displacement<Position<*>>, Any

Given an environment and a list of list of numbers, it creates a list of the right position type for the environment.
class CloseToGPSTrace<T> : AbstractCloseTo<T, GeoPosition>

This Displacement displaces nodes in the proximity of a GPS trace. Given a time interval from some time to another, it creates a TraceLoader, then uses the points in the interval to generate the sources for a Gaussian bivariate function and uses its probability density to deploy. Higher variance spreads nodes farther away from the trace with higher probability.
class GeometricGradientRectangle<P : Position<P>?> : Rectangle<P>

Distributes nodes geometrically within a rectangular shape.
class Point<P : Position<P>?> : Displacement<P>, Any

A single node in a single point.

class Circle<P : Position<P>?> : AbstractRandomDisplacement<P>

Distributes nodes in the first positions of it.unibo.alchemist.model.interfaces.GPSTrace.
