  • Interface Summary 
    Interface Description
    Wormhole2DA Wormhole (in this context) is an entity that "connects" two worlds: the "environment" and the "view".
    SlideInputManagerISlideInputManager is the base type for any class whose aim is to handle the the sliding of any physical/virtual device/control.
    AngleManagerA class that implements the IAngleManager interface is able to convert the sliding of any physical/virtual device/control into a positive double value that represents an angle.
    ZoomManagerA class that implements the IZoomManager interface is able to convert the sliding of any physical/virtual device/control into a positive double value that represents a zoom rate.
    ViewPortThis interface implements an Adapter pattern between a generic view element and the needs of a Wormhole2D.
    PointerSpeedBase type for any pointing device: it provides services to analyze the pointer's movement.
