Class BiochemistrydslBaseVisitor

  • All Implemented Interfaces:
    it.unibo.alchemist.biochemistrydsl.BiochemistrydslVisitor , org.antlr.v4.runtime.tree.ParseTreeVisitor 
    
public class BiochemistrydslBaseVisitor<T>
extends AbstractParseTreeVisitor<T> implements BiochemistrydslVisitor<T>

    This class provides an empty implementation of BiochemistrydslVisitor, which can be extended to create a visitor which only needs to handle a subset of the available methods.
