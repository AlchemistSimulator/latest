An [ImageEnvironment](../-image-environment/index.md) providing an [Euclidean2DNavigationGraph](../../it.unibo.alchemist.model.interfaces.geometry.euclidean2d.graph/index.md#-513689941%2FClasslikes%2F-267951372). The positions where to plant initial seeds should be specified directly in the image, marking each area of the environment with one or more pixels of a given color (defaults to blue).
